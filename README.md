# Auctionet Embedd

## Quick start
Add the script file in the ```<head>``` section.

```
  <script src="js/auctionet-embedd.js"></script>
```

Then add the embedd code wherever you want the widget:

```
  <script id='auctioNetEmbedd'>
    auctioNet.init({
      companyId: 52,
      locale: 'de'
    });
  </script>
```
