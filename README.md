# Auctionet Embed

## Quick start
Add the script file in the ```<head>``` section.

```
  <script src="js/auctionet-embed.1.0.min.js"></script>
```

Then add the embedd code wherever you want the widget:

```
  <script id='auctioNetEmbed'>
    auctioNet.init({
      companyId: 52,
      locale: 'de'
    });
  </script>
```
